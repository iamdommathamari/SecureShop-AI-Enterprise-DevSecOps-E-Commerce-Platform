# SecureShop-AI-Enterprise-DevSecOps-E-Commerce-Platform

SecureShop AI is a complete enterprise-grade online shopping platform similar to Amazon. It uses React for the website, Java Spring Boot for business logic, Python for AI features, PostgreSQL for data storage, and a full DevSecOps pipeline with Docker, Kubernetes, Jenkins, automated security scanning, monitoring, and cloud deployment to demonstrate real-world software engineering practices.
