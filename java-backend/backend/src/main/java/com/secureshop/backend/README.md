# Backend Java Package

This package contains the core backend Java code for SecureShop.

## Contents
- `BackendApplication.java` - Spring Boot application entry point.
- `config/` - Application configuration classes.
- `dto/` - Data transfer objects used for API requests and responses.
- `exception/` - Custom exceptions and global exception handling.
- `mapper/` - Mapper classes converting between DTOs and entities.
- `product/` - Product domain module (entity, controller, service, repository).
- `category/` - Category domain module (entity, controller, service, repository).

## Purpose
This package defines the backend domain logic and API surface for managing products and categories.
