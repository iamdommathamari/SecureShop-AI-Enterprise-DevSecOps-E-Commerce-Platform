# Test Package

This package includes backend unit and slice tests for SecureShop.

## Contents
- `BackendApplicationTests.java` - Basic Spring Boot test for application context loading.
- `category/` - Category service and controller test coverage.
- `product/` - Product service, repository, and controller test coverage.

## Purpose
- Validates API behavior and service logic with Spring MockMvc and Mockito.
- Ensures core modules are covered before running integration or end-to-end tests.
